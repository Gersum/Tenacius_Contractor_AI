# Pair Day 1 — Artifacts and Guidance

This folder contains the deliverables for Pair Day 1. Keep files here so reviewers and graders can read the morning-call sharpening, final question, explainer, and signoff together.

Recommended files (should live in this folder):

- `question.md` — Final committed question.
- `morning_call_summary.md` — Notes showing how the question was sharpened.
- `explainer.md` — Blog-style explainer that closes the asker's gap.
- `thread.md` — Tweet thread that summarizes the explainer.
- `sources.md` — The two canonical sources that the explainer relies on.
- `signoff.md` — Asker's confirmation that the gap is closed.

Quick rubric checklist for authors/reviewers:

- Question Sharpness: names a specific, actionable gap tied to a portfolio artifact.
- Morning-call sharpening: shows interrogation and concrete changes to the draft.
- Explainer Depth: names the load-bearing mechanism, cites two canonical sources, includes runnable example or concrete demonstration, and scopes the problem tightly.
- Public readiness: content is attributable, clean, and ready to publish under the author's name.
