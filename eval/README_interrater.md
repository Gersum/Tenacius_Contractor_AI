Inter-rater relabel README

This file accompanies `docs/inter_rater_assignment.md` and provides quick instructions for evaluators who will label the sampled tasks.

- Open `eval/inter_rater_sample.csv` or the per-rater CSV provided by the coordinator.
- Fill the binary columns with `1` for pass, `0` for fail, and use `comments` for short rationale.
- Return your CSV to the coordinator by the deadline.
